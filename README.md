# Chicha Clips 

Aplicación de escritorio para detectar y extraer automáticamente momentos destacados de grabaciones de videojuegos.

## ¿Qué hace?

- Analiza el audio de tu grabación con Whisper (OpenAI)
- Detecta momentos importantes según tus palabras clave
- Corta los clips automáticamente
- Muestra una galería de resultados

## Tecnologías

- Python
- PySide6
- OpenAI Whisper
- GPT-4o
- FFmpeg
- OpenCV

## Uso

1. Abre la aplicación
2. Selecciona tu video
3. Escribe el juego y las palabras clave de tu caller
4. Haz clic en "Analizar Video"
5. Revisa los clips detectados
